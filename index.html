Projeto Social

Página web interativa que simula um smartphone exibindo diferentes redes sociais, desenvolvida para praticar navegação entre páginas e organização de layouts com HTML e CSS.

Saiba mais...

Este projeto foi desenvolvido durante meus estudos em desenvolvimento web com o objetivo de praticar a criação de interfaces interativas utilizando HTML e CSS.

A proposta foi simular a interface de um smartphone, permitindo navegar entre diferentes redes sociais dentro da própria página. Para isso, foram utilizados recursos como iframes, links de navegação e estilização com CSS, criando a sensação de estar utilizando um aplicativo dentro de um celular.
